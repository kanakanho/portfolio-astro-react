# Astro React Template

- Astro
  - Vite
  - React
- Biome
- Storybook

# Getting Started

## 開発用サーバーの起動
```sh
$ yarn dev
```

## ビルド
```sh
$ yarn build
```

## プレビュー
```sh
$ yarn preview
```

## Linter & Formatter
```sh
$ yarn biome
```

## Storybook
```sh
$ yarn storybook
```

> [!WARNING]
> Storybook は React コンポーネントでのみ利用が可能です
